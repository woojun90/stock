## 数据流

```
┌──────────────────────────────────────────────────────────────────────────────────┐
│                              Weekly Rebalance Data Flow                           │
├──────────────────────────────────────────────────────────────────────────────────┤
│                                                                                   │
│  触发: 周五 14:54 (scheduled_rebalance)                                           │
│                                                                                   │
│  ┌────────────────────────────────────────────────────────────────────────────┐  │
│  │  1. market-data-snapshot                                                   │  │
│  │     输入: universe_codes, as_of, history_days                              │  │
│  │     输出: market_snapshot ───────────────────────────────────────────────┐ │  │
│  │     权限: read_only                                                      │ │  │
│  └──────────────────────────────────────────────────────────────────────────┼─┘  │
│                                                                               │    │
│  ┌────────────────────────────────────────────────────────────────────────────┐  │
│  │  2. momentum-ranking                                                       │  │
│  │     输入: market_snapshot ◀────────────────────────────────────────────────┘  │
│  │     输出: candidate_rankings ────────────────────────────────────────────┐   │  │
│  │     权限: state_write                                                    │   │  │
│  └──────────────────────────────────────────────────────────────────────────┼───┘  │
│                                                                               │      │
│  ┌────────────────────────────────────────────────────────────────────────────┐  │
│  │  3. risk-guard                                                             │  │
│  │     输入: market_snapshot, candidate_rankings ◀────────────────────────────┘  │
│  │     输出: risk_state, constraints ───────────────────────────────────────┐  │  │
│  │     权限: state_write (高优先级)                                          │  │  │
│  └──────────────────────────────────────────────────────────────────────────┼──┘  │
│                                                                               │     │
│  ┌────────────────────────────────────────────────────────────────────────────┐  │
│  │  4. market-context-interpreter (可选, AI)                                  │  │
│  │     输入: market_snapshot, candidate_codes                                 │  │
│  │     输出: market_advisory ─────────────────────────────────────────────┐   │  │
│  │     权限: advisory (不得修改确定性路径)                                  │   │  │
│  └──────────────────────────────────────────────────────────────────────────┼───┘  │
│                                                                               │       │
│  ┌────────────────────────────────────────────────────────────────────────────┐  │
│  │  5. portfolio-constructor                                                  │  │
│  │     输入: candidate_rankings ◀─────────────────────────────────────────────────┤
│  │            risk_state ◀───────────────────────────────────────────────────────┘
│  │     输出: target_portfolio ──────────────────────────────────────────────┐  │  │
│  │     权限: state_write                                                    │  │  │
│  └──────────────────────────────────────────────────────────────────────────┼──┘  │
│                                                                               │     │
│  ┌────────────────────────────────────────────────────────────────────────────┐  │
│  │  6. decision-explainer (可选, AI)                                          │  │
│  │     输入: decision_envelope                                                │  │
│  │     输出: human_readable_rationale ─────────────────────────────────────┐  │  │
│  │     权限: advisory                                                       │  │  │
│  └──────────────────────────────────────────────────────────────────────────┼──┘  │
│                                                                               │     │
│  ┌────────────────────────────────────────────────────────────────────────────┐  │
│  │  7. rebalance-executor                                                     │  │
│  │     输入: target_portfolio ◀──────────────────────────────────────────────────┘
│  │     输出: trade_proposal / trade_execute                               │     │
│  │     权限: trade_proposal (backtest/paper)                              │     │
│  │            trade_execute (live, 需审批)                                │     │
│  └────────────────────────────────────────────────────────────────────────────┘  │
│                                                                                   │
│  ─────────────────────────────────────────────────────────────────────────────── │
│                                                                                   │
│  最终输出: decision_envelope (聚合所有结果)                                       │
│                                                                                   │
└──────────────────────────────────────────────────────────────────────────────────┘
```

## AI 边界强制执行

```
┌──────────────────────────────────────────────────────────────────────────────────┐
│                              AI Guardrails Enforcement                            │
├──────────────────────────────────────────────────────────────────────────────────┤
│                                                                                   │
│   AI Skill 输出                                                                   │
│        │                                                                          │
│        ▼                                                                          │
│   ┌─────────────────────────────────────────────────────────────────────────┐   │
│   │                         Guardrails Check                                │   │
│   ├─────────────────────────────────────────────────────────────────────────┤   │
│   │                                                                          │   │
│   │   1. 检查 writable_fields                                                │   │
│   │      ├─ market_advisory         ✓ 允许                                  │   │
│   │      ├─ human_readable_rationale ✓ 允许                                 │   │
│   │      └─ 其他                    → 检查是否在 forbidden_fields            │   │
│   │                                                                          │   │
│   │   2. 检查 forbidden_fields                                               │   │
│   │      ├─ candidate_rankings      ✗ 拒绝                                  │   │
│   │      ├─ risk_constraints        ✗ 拒绝                                  │   │
│   │      ├─ target_portfolio        ✗ 拒绝                                  │   │
│   │      ├─ trade_proposal          ✗ 拒绝                                  │   │
│   │      └─ approval_state          ✗ 拒绝                                  │   │
│   │                                                                          │   │
│   │   3. 决策                                                                │   │
│   │      ├─ 全部通过     → 写入 context                                      │   │
│   │      ├─ 部分违规     → 拒绝违规字段，记录警告                            │   │
│   │      └─ 严重违规     → 拒绝整个输出，转人工复核                          │   │
│   │                                                                          │   │
│   └─────────────────────────────────────────────────────────────────────────┘   │
│        │                                                                          │
│        ▼                                                                          │
│   ┌─────────────────────────────────────────────────────────────────────────┐   │
│   │                         Audit Log                                        │   │
│   │   记录: AI输出内容、guardrails检查结果、最终决策                         │   │
│   └─────────────────────────────────────────────────────────────────────────┘   │
│                                                                                   │
└──────────────────────────────────────────────────────────────────────────────────┘
```

## 冲突优先级

```
┌──────────────────────────────────────────────────────────────────────────────────┐
│                            Conflict Resolution Priority                           │
├──────────────────────────────────────────────────────────────────────────────────┤
│                                                                                   │
│   优先级   来源              示例场景                                             │
│   ─────   ───────────────   ─────────────────────────────────────────────────   │
│                                                                                   │
│   1 (高)  risk-guard        风控要求禁止买入某ETF                                │
│           风控约束           → 无论ranking多高，都不得买入                       │
│                                                                                   │
│   2       policy_guardrail  策略规则限制某类别权重                               │
│           策略约束           → portfolio-constructor必须遵守                     │
│                                                                                   │
│   3       ai_advisory       AI建议降低仓位                                       │
│           AI建议             → 可参考，但不强制执行                              │
│                               → 如采纳需转人工确认                               │
│                                                                                   │
│   4 (低)  ranking_result    动量排名建议买入                                     │
│           排名结果           → 可被以上任意约束覆盖                              │
│                                                                                   │
│   ────────────────────────────────────────────────────────────────────────────   │
│                                                                                   │
│   规则: 高优先级约束覆盖低优先级建议                                              │
│                                                                                   │
└──────────────────────────────────────────────────────────────────────────────────┘
```
